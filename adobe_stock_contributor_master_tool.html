Panduan lengkap dari nol hingga first sale. Ikuti setiap step secara berurutan.
Fase 1 — Persiapan akun
1
Daftar sebagai Adobe Stock Contributor
Buka contributor.stock.adobe.com → klik "Join Now" → gunakan Adobe ID yang ada atau buat baru. Gratis, tidak butuh portfolio awal.
contributor.stock.adobe.com ↗
2
Lengkapi profil & tax info
Settings → Tax Information → isi W-8BEN (untuk WNI). Wajib sebelum bisa menerima pembayaran. Gunakan passport untuk verifikasi identitas.
3
Hubungkan metode pembayaran
Adobe membayar via PayPal atau Skrill. Minimum payout $25. Pembayaran tiap tanggal 15 bulan berikutnya.
Fase 2 — Persiapan file AI-generated
4
Wajib declare "AI-generated content"
Sejak 2024, Adobe wajibkan semua konten AI harus di-declare. Saat upload, centang opsi "Generative AI". Tidak comply = rejection + akun bisa ditangguhkan.
5
Hapus watermark & bersihkan file
Gunakan Adobe Firefly Generative Fill atau Cleanup.pictures untuk remove watermark Gemini/MJ. File harus bebas logo tools pihak ketiga.
6
Spesifikasi teknis file
Minimum 4MP untuk foto/ilustrasi. PNG dengan transparansi didukung untuk clipart/sticker. JPEG min quality 8/10. Max file size 45MB.
Fase 3 — Upload pertama (submission test)
7
Upload 1–5 file sebagai test batch
Jangan langsung upload ratusan file. Mulai dengan 3-5 file terbaik untuk lihat acceptance rate. Review biasanya 1-3 hari kerja.
8
Isi metadata dengan benar
Title: maks 200 karakter, deskriptif. Keywords: maks 50 keywords, pisahkan koma. Category pilih yang paling relevan. Gunakan tab Metadata Generator di atas!
✓
Monitor dashboard & iterasi
Setelah approved, track performa via Analytics. File dengan views tinggi tapi download rendah = perlu keyword improvement.

Adobe Stock API memungkinkan otomatisasi upload, batch metadata, dan monitoring performa portfolio.
1. Setup Adobe Developer Console
1
Buat project di Adobe Developer Console
Buka developer.adobe.com → My Projects → Create Project → Add API → Stock API. Pilih OAuth Server-to-Server credential.
developer.adobe.com/console ↗
2
Simpan credentials
Copy Client ID dan Client Secret. Simpan di environment variable, JANGAN hardcode di script. Gunakan .env file atau secret manager.
2. Endpoint penting
POST /api/stock/v3/files
Upload file baru ke contributor portal
PUT /api/stock/v3/files/{id}/metadata
Update title, keywords, category
GET /api/stock/v3/files?status=approved
List semua file yang approved
GET /api/stock/v1/stats/earnings
Earnings & download statistics
3. Contoh batch upload script (Python)
import os, requests API_KEY = os.getenv("ADOBE_STOCK_API_KEY") BASE_URL = "https://contributor.stock.adobe.com" def upload_file(filepath, title, keywords): headers = { "x-api-key": API_KEY, "Content-Type": "multipart/form-data" } with open(filepath, "rb") as f: files = {"file": f} data = { "title": title, "keywords": ",".join(keywords), "is_ai_generated": "true", "category": 1 } r = requests.post( f"{BASE_URL}/api/stock/v3/files", headers=headers, files=files, data=data ) return r.json() # Batch loop dari folder for img in os.listdir("./ready_to_upload"): if img.endswith(".png"): result = upload_file( f"./ready_to_upload/{img}", title=f"Neon cyber sticker — {img}", keywords=["neon","cyber","sticker","kawaii"] ) print(f"Uploaded: {result}")
4. Automation workflow yang direkomendasikan
A
Generate → Clean → Rename
Gemini/MJ generate → remove watermark → rename file dengan naming convention: [niche]_[style]_[number].png
B
Batch metadata via CSV
Buat CSV dengan kolom: filename, title, keywords, category. Import ke upload portal atau gunakan API untuk batch update.
C
Monitor & optimize weekly
Setiap minggu check: top performing files → replicate style. Low performer → update keywords atau archive.
